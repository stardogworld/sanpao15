# 三炮十五兵求解器 v1.0.0

这是项目的第一个正式版本。

## 主要内容

- 完整三炮十五兵规则引擎。
- 分层 WDL 表库支持。
- MTD 表库支持。
- MTD-aware 最优着法推荐。
- 本地 HTTP 后端。
- Web UI 分析界面。
- 表库 inspect / verify 工具。
- 支持 mmap 加载大型表库。

## 注意

完整表库文件体积较大，不包含在本 release 附件中：

- WDL 表库约 4.37 GiB。
- MTD 表库约 26.25 GiB。
- 合计约 30.62 GiB。

请自行生成或按项目文档配置表库目录。GitHub Release 默认只发布源码和说明，不上传 `.s15res`、`.s15mtd`、构建目录或日志文件。
