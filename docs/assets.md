# Assets

Current UI assets are self-created for this repository:

- `ui/src/assets/pieces/cannon.svg`
- `ui/src/assets/pieces/soldier.svg`

They are simple self-created Chinese SVG pieces rendered directly on the board
for high-DPI displays. The current pieces use large `炮` / `兵` characters with
project-defined colors and borders. No third-party piece set, web image, or
hotlinked asset is used.

Future asset changes must record:

- source URL or author;
- license;
- attribution text if required;
- whether modification is allowed;
- whether redistribution inside this repository is allowed.

Assets with unclear license terms should not be used.
